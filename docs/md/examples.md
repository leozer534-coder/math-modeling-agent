## 优化问题

### 案例 1：线性规划 - 生产计划优化

**问题描述:**

某工厂生产 A、B 两种产品，相关数据如下：

| 产品 | 原材料 1(kg) | 原材料 2(kg) | 工时 (小时) | 利润 (元) |
|------|-------------|-------------|------------|----------|
| A    | 2           | 1           | 3          | 5        |
| B    | 1           | 2           | 2          | 4        |

工厂每天可用资源：原材料 1 共 100kg，原材料 2 共 80kg，工时 120 小时。问如何安排生产计划使利润最大？

**使用方法:**

在聊天界面输入：
```
问题：某工厂生产 A、B 两种产品。
产品 A：需要原材料 1 为 2kg，原材料 2 为 1kg，工时 3 小时，利润 5 元/件。
产品 B：需要原材料 1 为 1kg，原材料 2 为 2kg，工时 2 小时，利润 4 元/件。
工厂每天可用资源：原材料 1 共 100kg，原材料 2 共 80kg，工时 120 小时。
问：如何安排 A、B 两种产品的生产数量，使得总利润最大？

请建立线性规划模型并求解。
```

**预期输出:**

1. 建立线性规划模型：
   - 决策变量：x₁ (A 产品数量), x₂ (B 产品数量)
   - 目标函数：max Z = 5x₁ + 4x₂
   - 约束条件：
     - 2x₁ + x₂ ≤ 100 (原材料 1)
     - x₁ + 2x₂ ≤ 80 (原材料 2)
     - 3x₁ + 2x₂ ≤ 120 (工时)
     - x₁, x₂ ≥ 0

2. Python 代码求解（使用 scipy 或 pulp）

3. 最优解：生产 A 产品 20 件，B 产品 30 件，最大利润 220 元

### 案例 2：整数规划 - 选址问题

**问题描述:**

某公司计划在 5 个候选城市中选择 3 个建立仓库，各城市的建设成本和覆盖区域如下：

| C    | 120           | 1, 3, 5  |
| D    | 130           | 2, 4, 5  |
| E    | 110           | 3, 4, 5  |

要求：必须覆盖所有 5 个区域，且总成本最小。


### 案例 3：非线性规划 - 投资组合优化

**问题描述:**

某投资者有 100 万元资金，可投资 3 种股票，预期收益率和协方差矩阵如下：

- 预期收益率：r = [0.08, 0.12, 0.10]
- 协方差矩阵：
```
Σ = [[0.04, 0.01, 0.02],
     [0.01, 0.09, 0.03],
     [0.02, 0.03, 0.06]]
```

要求在风险（方差）不超过 0.05 的前提下，最大化预期收益。

**使用方法:**

```
问题：投资组合优化
可用资金：100 万元
3 种股票的预期收益率：[8%, 12%, 10%]
协方差矩阵：
[[0.04, 0.01, 0.02],
 [0.01, 0.09, 0.03],
 [0.02, 0.03, 0.06]]

约束条件：
1. 投资比例之和为 1
2. 投资组合风险（方差）不超过 0.05
3. 不允许卖空（投资比例非负）

目标：最大化预期收益率

请建立非线性规划模型并求解。
```

**预期输出:**

1. 建立 Markowitz 投资组合模型
2. 使用 scipy.optimize.minimize 求解
3. 最优投资比例和预期收益

### 案例 12：微分方程模型 - 传染病传播预测

**问题描述:**

使用 SIR 模型预测传染病传播趋势。

**使用方法:**

```
问题：传染病传播 SIR 模型

SIR 模型：
- S(t): 易感人群
- I(t): 感染者
- R(t): 康复者

微分方程：
dS/dt = -β*S*I/N
dI/dt = β*S*I/N - γ*I
dR/dt = γ*I

参数：
- N = 100000 (总人口)
- β = 0.3 (感染率)
- γ = 0.1 (康复率)
- 初始：I(0) = 10, S(0) = N-10, R(0) = 0

请：
1. 建立 SIR 模型
2. 数值求解微分方程
3. 预测 100 天内的传播趋势
4. 分析不同β值对传播的影响
5. 讨论防控措施的效果
```

**预期输出:**

1. SIR 模型建立
2. 使用 scipy.integrate.odeint 求解
3. 传播曲线可视化
4. 参数敏感性分析
5. 防控建议
